Summazer

Summazer è una applicazione web in grado di "spremere" un testo o una pagina web ed estrarne il succo, ovvero le frasi con più alto contenuto informativo per generare un riassunto del testo elaborato.

Web Services

SOAP SEO Web Services: le applicazioni di text-mining disponibili su NelSenso possono essere richiamate anche attraverso SOAP Web Services dai vostri CMS (tags: TITLE, Description, Keywords, Tags ...) per una efficace posizionamento delle vostre pagine web sui motori di ricerca.

IRezer

IRezer: il tool di Information Retrieval (IR) che consente l'estrazione automatica delle keywords (Tags) e delle frasi "chiave" più semanticamente rilevanti in un testo o pagina web.

IRezer

Clustezer

Clustezer è un tool che permette la classificazione automatica di keywords e keyprhases di una pagina web, in categorie/aree semantiche auto-determinate.
